# Te Amo Restaurant

> Taste of Togetherness — Fine dining, AC banquets & wedding lawns in Ahilyanagar, Maharashtra.

A modern React + Vite + TypeScript + Tailwind CSS website for Te Amo Restaurant.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (styling)
- **shadcn/ui** (component library)
- **React Router v6** (routing)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deploy on Vercel

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Framework auto-detected as **Vite**
4. No environment variables needed
5. Click Deploy ✓

The `vercel.json` handles SPA routing automatically.

## Pages

- `/` — Home
- `/menu` — Full menu
- `/banquet` — Banquet & events
- `/gallery` — Photo gallery
- `/contact` — Contact & reservation form
